# Speech-Emotion-Recognition-BLSTM-Attention

## Model  
LLDs (32 dim) + BLSTM + Attention (ref1)

## DataSet  
Japanese Twitter-based Emotional Speech (JTES)

## Usage  
1. Edit Features/extract.py and set your path. Then, do "python3 extract.py"
2. "python3 test.py"
